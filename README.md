The LittleBlackBook project is meant to create an Agentic Product Manager that will automate all tasks of a Product Manager. Currently, given a set of functionality requirements and a given budget, it will first check for its feasibility and then it will try and maximize the value by splitting the budget between marketing and product functions and also choosing the set of functioanlity judiciously. It is a jupyter notebook that can also be run or Google Colab.
